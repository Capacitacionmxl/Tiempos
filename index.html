Guía para evaluar Tiempos
           
    Descripción Tolerancia Variable/ subgrupo afectada Descripción
Inicio de interacción 1 Mencionar Script de Entrada al iniciar la interacción 3 Segundos Amabilidad Si el RAC se presenta después de 3 segundos y el cliente NO hace referencia a que estuvo esperando, se afecta Amabilidad.
Apego Si el RAC se presenta después de 3 segundos y el cliente hace referencia a que estuvo esperando, se afecta Semaforo de Actitud en Rojo y se quita Amabilidad y Apego y se marca como Tiempo de espera injustificado.
SOLUCIÓN/ACTITUD Si el cliente cuelga debido a que el RAC tardó más de 3 segundos en presentarse, se afecta se clasificará como Mala Práctica.
2 El RAC se presenta pero el cliente no responde 20 Segundos Apego Tras presentarse, el RAC debe hacer 2 intentos de comunicarse con el cliente (uno aproximadamente cada 10 segundos) y despedirse.
SOLUCIÓN/ACTITUD El RAC debe liberar la llamada antes de 25 segundos, si demora más se considera Mala práctica y se marca como Tiempo de espera injustificado.
Durante la interacción 3 Abandono al usuario: 3 segundos Amabilidad Se penalizará la variable de Amabilidad
Dejar en espera al usuario sin previo aviso 30 segundos Apego y Amabilidad Se penalizará la variable Apego y Amabilidad se convierte en Semáforo Rojo aunque el cliente no cuelgue.
SOLUCIÓN/ACTITUD En caso de que el usuario cuelgue se considera Mala Práctica.
4 Pruebas innecesarias: Al realizarse pruebas que no correspondan al flujo de solución según el sondeo. 2 Pruebas Apego En caso de Solucionar o Escalar de forma correcta.
SOLUCIÓN/ACTITUD En caso de No Solucionar; se considerará Mala Práctica. (SLA)
5 Preguntas repetitivas Información redundante a) Cuando el cliente debe repetir un dato por falta de atención del RAC. Apego/Empatía Si el cliente externa molestia.
b) Cuando el RAC menciona más de una vez algún dato/proceso sin que el cliente lo solicite. Apego, Empatía

Correcta (de Esc)		 Si el RAC continua solicitando datos o información que ya había solicitado, se interveniene la llamada y se penaliza en Semáforo Rojo.
6 Al finalizar llamada el cliente se despide pero no cuelga bien el teléfono. 15 segundos Amabilidad Dentro de la tolerancia el RAC debe despedirse de nuevo antes de liberar la llamada. Si no se despide de nuevo, se afecta Amabilidad .
Apego Si el RAC rebasa la tolerancia sin liberar la llamada se afecta Apego.
ACTITUD Si rebasa los 30 s, se convierte en Mala Práctica.
7 En transferencia.
A partir que de que el RAC se despide.		 5 segundos Apego Al rebasar la tolerancia para realizar la transferencia, se penalizará Apego.
10 segundos ACTITUD Al rebasar la tolerancia se penalizará la variable ACTITUD.
A solicitud del cliente 8 Si el cliente se ausenta de la llamada sin solicitar que se le espere 1 minuto Apego Al cumplirse 1 min el RAC debe intentar establecer comunicación con el cliente, si este no responde el RAC tiene 15 s para liberar la llamada. (2:15 minutos máximo).
3 intentos (1 cada 20 s, aprox) Amabilidad Si el RAC no intenta retomar la comunicación, se afecta Amabilidad.
9 Si el cliente se ausenta de la llamada pero solicita que se le espere 1 frase del cliente Amabilidad
Empatía		 El RAC debe estar atento y responder de manera inmediata en cuanto el cliente retome la llamada; se afecta Amabilidad y Empatía si responde al segundo intento del cliente.
Apego
Amabilidad		 Si el RAC responde al tercer o más intentos del cliente, pero el cliente no se queja, también se afectará Apego.
ACTITUD Si el RAC responde al tercer o más intentos del cliente, pero el cliente se queja, también se afectará ACTITUD y se considera Mala Práctica.
3 minutos Apego Al cumplirse 3 min el RAC debe intentar establecer comunicación con el cliente, si este no responde el RAC tiene 15 s para liberar la llamada (4:15 minutos máximo).
3 intentos (1 cada 20 s, aprox) Amabilidad/Empatía El RAC debe realizar 3 intentos (uno cada 20 segundos) para tratar de establecer comunicación de nuevo con el cliente, si no lo hace se afecta Amabilidad.
10 Si el cliente se ausenta de la llamada porque se le solicitó hacer pruebas 3 minutos Apego Al cumplirse 3 min el RAC debe intentar establecer comunicación con el cliente, si este no responde el RAC tiene 15 s para liberar la llamada; si rebasa este tiempo se considera Mala Práctica (4:15 minutos máximo).
3 intentos (1 cada 20 s, aprox) Amabilidad/Empatía El RAC debe realizar 3 intentos (uno cada 20 segundos) para tratar de establecer comunicación de nuevo con el cliente, si no lo hace se afecta Amabilidad y Empatía.
11 Tiempo de espera justificado:
Cuando se deja al cliente debido a que un proceso implica que el RAC se mueva de su posición o debe solicitar apoyo.
(Se debe retomar cada 2:30 minutos)		 ≤ 30 segundos Apego Si el RAC se ausenta sin que el proceso que está realizando lo requiera.
> 30 segundos ACTITUD
Hasta 2:30 minutos Apego Si pasan más de 2:30 minutos sin que el RAC retome.
Después de 2:30 Apego
Empatía		 Si el cliente no es atendido después de tres frases o interjecciones buscando llamar la atención del RAC.
SOLUCIÓN/ACTITUD En caso de que el usuario cuelgue se considera Mala Práctica.
           
Consideraciones
           
  1 En todos los casos que se indique que el RAC debe liberar la llamada éste deberá notificar a un Supervisor para que lo apruebe y registre el evento en una bitácora.
  2 Antes de que el RAC libere una llamada empleará el script homologado que corresponda.
